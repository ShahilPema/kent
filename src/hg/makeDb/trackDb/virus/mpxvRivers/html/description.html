Copy this entire assembly hub for local use

This download is only for the purpose of using this assembly hub in your institution which may have firewall access restrictions to this data.
To download this assembly data, use this rsync command: 

  rsync -a -P \
    rsync://hgdownload.soe.ucsc.edu/hubs/GCF/014/621/545/GCF_014621545.1/ \
      ./GCF_014621545.1/
which creates the local directory: ./GCF_014621545.1/
or this wget command: 
  wget --timestamping -m -nH -x --cut-dirs=6 -e robots=off -np -k \
    --reject "index.html*" -P "GCF_014621545.1" \
       https://hgdownload.soe.ucsc.edu/hubs/GCF/014/621/545/GCF_014621545.1/
which creates a local directory: ./GCF_014621545.1/

There is an included hub.txt file in that download data directory to use for your local track hub instance.

select the My Hubs tab to enter a URL to this hub.txt file to attach this assembly hub to a genome browser.

blat service

There is blat service available for this genome assembly. When viewing this assembly in the genome browser, access the blat service via the Tools -> Blat blue navigation bar menu item.

For local command line blat service, access the blat service via the gfClient command line operation.
See also: hgdownload.soe.ucsc.edu/admin/exe/linux.x86_64/ to download command line binaries.

To operate this locally, you will need the GCF_014621545.1.2bit file from: 

  https://hgdownload.soe.ucsc.edu/hubs/GCF/014/621/545/GCF_014621545.1/
Which can be obtained with rsync via: 
  rsync -a -P 
    rsync://hgdownload.soe.ucsc.edu/hubs/GCF/014/621/545/GCF_014621545.1/GCF_014621545.1.2bit ./
With that GCF_014621545.1.2bit file in your working directory where you run this command, for example, a DNA query with your DNA sequence in the file: someDna.fa with result in the file: GCF_014621545.1.someDna.psl 
gfClient -t=dna -q=dna -genome=GCF_014621545.1 -genomeDataDir=GCF/014/621/545/GCF_014621545.1 
    dynablat-01.soe.ucsc.edu 4040 ./ someDna.fa GCF_014621545.1.someDna.psl
For a protein fasta query with your protein sequence in the file: someProtein.faa with result in the file: GCF_014621545.1.someProtein.psl 
gfClient -t=dnax -q=prot  -genome=GCF_014621545.1 -genomeDataDir=GCF/014/621/545/GCF_014621545.1 
    dynablat-01.soe.ucsc.edu 4040 ./ someProtein.faa GCF_014621545.1.someProtein.psl
